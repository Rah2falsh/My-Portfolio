# Rahaf Alshammari — Portfolio

A personal portfolio website built from scratch using HTML, CSS, and JavaScript — no frameworks, no libraries.

---

## About

I'm a recently graduated Software Engineer with a focus on UI/UX design and frontend development. The portfolio reflects how I work: I think like a designer and build like a developer — from user research all the way to production-ready code.

The site showcases three projects across different disciplines, along with my background, skill set, and contact information.

---

## Projects

**SKILLUP** — An internal training platform that connects learning to job performance. Built with Figma, React, and Firebase.

**NAQDI** — A digital banking UI concept focused on simplifying financial operations and improving experience clarity. Designed in Figma with a complete design system.

**HEALTHY** — Health tracking app interfaces for steps, calories, and vitals. Designed to feel simple and fluid.
